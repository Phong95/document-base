## Request Model

| Name           | Description                   | Required                                        | Type    |
| -------------- | ----------------------------- | ----------------------------------------------- | ------- |
| GrantType      | `password` or `refresh-token` | <Highlight color="#EE4B2B">Required</Highlight> | string  |
| UserName       |                               |                                                 | string  |
| RefreshToken   |                               |                                                 | string  |
| Password       |                               |                                                 | string  |
| RememberMe     |                               |                                                 | boolean |
| RecaptchaToken |                               |                                                 | string  |

## Response

| Code                           | Description        |
| ------------------------------ | ------------------ |
| <Dot color="#25c2a0">200</Dot> | Success.           |
| <Dot color="#FFC300">401</Dot> | Unauthorized.      |
| <Dot color="#FFC300">404</Dot> | Not Found.         |
| <Dot color="#EE4B2B">500</Dot> | Error From Server. |

## Response Model

| Name                   | Description | Type    |
| ---------------------- | ----------- | ------- |
| UserEmail              |             | string  |
| AccessToken            |             | string  |
| ExpiresAt              |             | Date    |
| IsAdmin                |             | boolean |
| IsBanned               |             | boolean |
| RefreshTokenExpiration |             | string  |
| RefreshToken           |             | string  |
| Role                   |             | string  |
