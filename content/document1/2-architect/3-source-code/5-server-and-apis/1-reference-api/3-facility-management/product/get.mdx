`https://domain/api/fm-mobile/v1/c-{clientId}/p-{projectId}/products/single/{productId}`

## Request Parameters

| Name      | Description | Required                                        | Type   |
| --------- | ----------- | ----------------------------------------------- | ------ |
| clientId  |             | <Highlight color="#EE4B2B">Required</Highlight> | string |
| projectId |             | <Highlight color="#EE4B2B">Required</Highlight> | string |
| productId |             | <Highlight color="#EE4B2B">Required</Highlight> | string |


## Response

| Code                           | Description        |
| ------------------------------ | ------------------ |
| <Dot color="#25c2a0">200</Dot> | Success.           |
| <Dot color="#FFC300">401</Dot> | Unauthorized.      |
| <Dot color="#FFC300">404</Dot> | Not Found.         |
| <Dot color="#EE4B2B">500</Dot> | Error From Server. |
